# vscode

* [√] 查看所以已安装组件
* [√] 查看指定 group 安装信息(已安装，版本有变化,未安装)
* [√] 安装指定 group 组件列表 (-f 强制更新版本不一致组件)
* [-] 安装指定 extension 组件 (-f 强制更新版本不一致组件)

## vscode list extension

```bash
code --list-extensions
code --list-extensions --show-versions

code --install-extension $extension

```
