# dotfiles

My Dev Tool Environment - Dotfiles

* 在 debian,ubuntu 等系列 linux 系统上安装(cpp,python,rust,...) 相关开发空间及vscode等插件。
* debian,ubuntu

本项目默认安装到  `/opt/tong/` 路径；

```bash
mkdir -p /opt/tong
cd /opt/tong && git clone git@github.com:Ts-sound/dotfiles.git
```
